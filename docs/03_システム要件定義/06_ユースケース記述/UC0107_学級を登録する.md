# UC0107_学級を登録する

## パッケージ
- 年間データ登録

## アクタ
- 教務

## 事前条件
- 教員、科目、教室がシステムに登録されている。

## 事後条件
- 学級がシステムに登録されている。

## 開始トリガ
- `学級情報入力モーダル`が表示される。

## 基本フロー
1. 教務は、学級名、担任、開講曜日、コマ数を入力する。
2. システムは、入力値のチェックを行う。
3. システムは、学級を登録する。
4. システムは、`学級情報入力モーダル`を閉じる。

## 代替フロー
2a. 入力が不正なとき
1. システムは、エラーメッセージを表示する。→基本フロー1へ戻る。
