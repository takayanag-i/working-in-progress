# UC0102_教員を登録する

## パッケージ
- 年間データ登録

## アクタ
- 教務

## 事前条件
- 教務がシステムにログインしている。

## 事後条件
- 教員がシステムに登録されている。

## 開始トリガ
- 教務が`教員管理画面`で任意に開始する。

## 基本フロー
1. システムは、`教員情報入力モーダル`を表示する。
2. 教務は、追加する教員の教科と表示名、持ちコマ数を入力する。
3. システムは、入力値のチェックを行う。
4. システムは、教員を登録する。
5. システムは、`教員情報入力モーダル`を閉じる。

## 代替フロー
4a. 入力値が不正なとき
1. システムは、エラーメッセージを表示する。→基本フロー2へ戻る。
