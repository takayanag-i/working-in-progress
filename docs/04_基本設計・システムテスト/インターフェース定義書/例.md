# 講座編成

## オブジェクト構造

### 講座編成

| キー名       | データ型  | 必須性 | 説明                                |
|--------------|-----------|--------|-------------------------------------|
| homeroom     | String    | 必須   | 学級名                              |
| blocks       | Array\<Object\>      | 必須   | 1つ以上のブロックオブジェクトを格納 |

### ブロック

| キー名       | データ型  | 必須性 | 説明                                |
|--------------|-----------|--------|-------------------------------------|
| name    | String    | 必須   | ブロック名                          |
| lanes        | Array\<Object\>      | 必須   | 1つ以上のレーンオブジェクトを格納 |

### レーン

| キー名       | データ型  | 必須性 | 説明                                |
|--------------|-----------|--------|-------------------------------------|
| count     | Number    | 必須   | レーン連番                            |
| courses      | Array\<String\>      | 必須   | 1つ以上の講座IDを格納   |

---

## JSONデータ例

```json
{
  "homeroom": "2-4",
  "blocks": [
    {
      "name": "2-4共通",
      "lanes": [
        {
          "count": 1,
          "courses": ["数学Ⅱ", "物理基礎"]
        },
      ]
    },
    {
      "name": "2-4歴史",
      "lanes": [
        {
          "count": 1,
          "courses": ["世界史探求"]
        },
        {
          "count": 2,
          "courses": ["日本史探求"]
        }
      ]
    }
  ]
}
