# UC0212_制約条件の内容を変更する（組み込み）

## パッケージ
- 年間時間割編成

## アクタ
- 教務

## 事前条件
- 組み込み制約条件がシステムに登録されている。

## 事後条件
- 制約条件の内容が更新されている。

## 開始トリガ
- `組み込み制約設定モーダル`が表示される。

## 基本フロー
1. 教務は、制約タイプに応じた項目の変更内容を入力する。
2. システムは、入力値のチェックを行う。
3. システムは、制約条件を更新する。
4. システムは、`組み込み制約設定モーダル`を閉じる。

## 代替フロー
2a. 入力が不正なとき
1. システムは、エラーメッセージを表示する。→基本フロー1へ戻る。
